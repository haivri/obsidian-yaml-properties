# Contributing

Bug reports and pull requests are welcome.

Before submitting a change:

1. Run `npm install`.
2. Run `npm run build`.
3. Test Live Preview, Source mode, and Reading view in both light and dark themes.
4. Test with **Properties in document** set to both **Visible** and **Source**.
5. Do not include vault content or `data.json` in commits.

Keep the plugin local-first: new functionality must not transmit note content without explicit user action and clear documentation.
